## Project Description
This is an Android-based Expense Tracking application developed using Java and Android Studio.  
The app allows users to log in, add expenses, and switch between light and dark themes.

## Features
- Login authentication
- Add daily expenses
- Dark / Light mode
- Simple and user-friendly UI

## Tools & Technologies
- Android Studio
- Java
- XML
- Material Design

## How to Run
1. Clone or download the project
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or physical device
